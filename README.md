# "Análisis de un programa de fidelización de una aerolínea"
En este proyecto analizo el comportamiento de clientes de un programa de lealtad de una aerolínea, a partir de dos datasets (actividad de vuelos y perfil de clientes), para sacar conclusiones sobre estacionalidad, correlaciones, perfil de cliente, etc.

# Datos
El primer data set "Flights Activity" nos proporciona la informacion de cada cliente y su actividad de vuelo (reservas, distancia, puntos de fidelizacion..)
Y el segundo data set "Loyalty History" nos informa directamente de los detalles de cada cliente (educacion, salario, estado civil..)

# Fases
1. En la primera fase realizamos una exploracion de los datos y así familiarizarnos con la informacion que tenemos y revisar posibles nulos, duplicados y hacer esa limpieza de los datos para así poder trabajar mejor con ellos y evitar errores y sesgos de informacion.
2. En la segunda fase procedemos a hacer un analisis de la informacion (describe, outliers, correlación, frecuencias).
3. En la tercera fase visualizamos tanto variables numericas y categoricas para comprobar la relacion entre ellas y sacar conclusiones utiles para la aerolinea a la hora de conocer a los clientes y saber que mejor servicio ofrecerles.
4. En la cuarta fase evaluamos las diferencias en reservas de vuelo segun en nivel educativo para poder confirmar si es relevante a la hora hacer reservas.

# Herramientas
Python, pandas, numpy, matplotlib, seaborn, Jupyter.
